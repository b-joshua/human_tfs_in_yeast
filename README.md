# Analysis Code for Human TF Binding in Yeast Genome

This repository contains the analysis code and supporting files for the study:

**Beyond motif recognition: Specificity of human transcription factors in yeast**  
Joshua Bugis, Dan Reuben Zlotnik-Weinberg, Wajd Manadre, Vladimir Mindel, Yunwei Lu, Juan Ignacio Fuxman Bass, Naama Barkai

All scripts are organized to allow reproduction of the main analyses and figures from the paper.  

## Overview

This project investigates how human transcription factors (TFs) bind across the budding yeast genome, focusing on determinants of binding specificity beyond canonical DNA-binding motifs.

## Repository Contents

- `code/` – Core scripts and pipelines used for data processing and analysis  
- `metadata/` – Contains strain sequence information and data from additional completed analyses
- `sumproms/` – Promoter-level ChEC-seq binding scores of human TFs across the yeast genome
